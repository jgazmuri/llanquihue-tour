# LlanquihueTourApp

## Descripción
Sistema de gestión de tours para la agencia Llanquihue Tour, ubicada en la Región de Los Lagos. 
Permite cargar tours desde un archivo .txt y mostrarlos filtrados por tipo y precio.

## Estructura de carpetas
LlanquihueTourApp/
├── src/
│   ├── model/
│   │   └── Tour.java
│   ├── data/
│   │   └── GestorDatos.java
│   └── ui/
│       └── Main.java
└── resources/
    └── tour.txt

## Instrucciones para ejecutar
1. Abrir el proyecto en IntelliJ IDEA
2. Verificar que el archivo resources/tour.txt existe
3. Ejecutar la clase Main.java ubicada en el paquete ui
4. La consola mostrará todos los tours, filtrados por tipo y por precio
